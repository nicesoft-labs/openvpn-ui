Действия перезапуска

{{template "common/alert.html" .}}
Перезапустить сервер   Условный перезапуск сервера OpenVPN применит новую конфигурацию и переинициализирует все активные клиентские сессии.
Перезапустить контейнер   Полный перезапуск контейнера OpenVPN для повторного применения правил фаервола и перезагрузки сервера OpenVPN. Занимает больше времени, чем условный перезапуск.
Перезапустить UI   Перезапуск контейнера OpenVPN-UI. Текущая веб-сессия будет отключена.

Действия удаления

ОПАСНАЯ ЗОНА! Вы не сможете откатить действия из этого раздела!
Удалить PKI   Удалить PKI, все ключи сервера OpenVPN и клиентские сертификаты. Это не позволит клиентам подключаться с текущими сертификатами!
Удалить .ovpn   Удалить файлы "*.ovpn" из каталога "/etc/openvpn/clients".
Удалить статические   Удалить все резервации статических IP из каталога "/etc/openvpn/staticclients".
УДАЛИТЬ ВСЁ   Эта кнопка выполнит все шаги выше по порядку — удалит PKI, ключи сервера, клиентские сертификаты и файлы "*.ovpn".
Сброс БД   Удалить базу данных OpenVPN-UI. После этого перезапустите OpenVPN-UI. База будет создана заново при следующем запуске контейнера OpenVPN UI.

Действия инициализации

ОПАСНАЯ ЗОНА! Вы не сможете откатить действия из этого раздела!
Будьте осторожны, необходима соответствующая квалификация! Краткое руководство EasyRSA, Подробная документация EasyRSA.
EasyRSA VARs   Скопировать переменные EasyRSA из "./openvpn/config/easy-rsa.vars" в "./easy-rsa/pki/vars".
Generate CRL   Сгенерировать новый список отозванных сертификатов (CRL), выполнив "easyrsa gen-crl".
Initialize PKI   Инициализировать PKI, выполнив "easyrsa init-pki".
Build CA   Создать центр сертификации (CA), выполнив "easyrsa build-ca nopass".
Build Server   Сгенерировать новый серверный сертификат, ключ и файлы шифрования, выполнив "easyrsa gen-req & sign-req server nopas".
Generate DH   Сгенерировать новый ключ Диффи-Хеллмана, выполнив "easyrsa gen-dh". Это может занять до 10 минут. Ничего не зависло, дождитесь завершения.
Generate TA   Сгенерировать новый TA-ключ OpenVPN, выполнив "openvpn --genkey --secret ta.key".
INITIALIZE ALL   Эта кнопка выполнит все шаги выше по порядку — скопирует переменные EasyRSA и создаст PKI, CA, сервер, DH, TA и CRL.
