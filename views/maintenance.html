Restart Actions

{{template "common/alert.html" .}}
Restart Server   Conditional OpenVPN Server restart will apply new configuration and re-initiate all active clients sessions.
Restart Container   Full OpenVPN container restart to re-apply Firewall rules and reload OpenVPN Server. Takes more time than Conditional Restart.
Restart UI   OpenVPN-UI Container Restart. Current Web session will be disconnected.

Remove Actions

DANGER ZONE! You can't rollback any actions of this section!
Delete PKI   Delete PKI, all OpenVPN Server keys, and Client certificates. This will prevent clients from connecting using their current certificates!
Delete .ovpn   Delete "*.ovpn" files from the "/etc/openvpn/clients" directory.
Delete Static   Delete all static IP reservations from the "/etc/openvpn/staticclients" directory.
DELETE ALL   This button will execute all the seteps above, one by one - delete PKI, Server keys, Client certificates and "*.ovpn".
Drop DB   Drop the OpenVPN-UI Database. Restart OpenVPN-UI after. Database will be re-created during next OpenVPN UI container start.

Initialization Actions

DANGER ZONE! You can't rollback any actions of this section!
Be cautious, expertise level is essential! EasyRSA Quick Guide, EasyRSA detailed Guide.
Памятка по инициализации (рус.)
Выполните действия ниже последовательно:
  1. Скопируйте переменные EasyRSA (кнопка EasyRSA VARs).
  2. Сгенерируйте новый список отозванных сертификатов (CRL).
  3. Инициализируйте PKI.
  4. Создайте центр сертификации (CA).
  5. Сгенерируйте серверный сертификат и подпишите запрос.
  6. Сгенерируйте ключ Диффи-Хеллмана.
  7. Сгенерируйте TA-ключ OpenVPN.
  8. Кнопка INITIALIZE ALL выполнит все шаги по порядку автоматически.
EasyRSA VARs
{{with .pkiStatus.Vars}} {{if .Exists}}Создано{{else}}Создайте{{end}} {{.Path}} {{end}}
  Copy EasyRSA vars from "./openvpn/config/easy-rsa.vars to ./easy-rsa/pki/vars".
Generate CRL
{{with .pkiStatus.CRL}} {{if .Exists}}Создано{{else}}Создайте{{end}} {{.Path}} {{end}}
  Generate a new Certificate Revocation List (CRL) by running "easyrsa gen-crl".
Initialize PKI
{{with .pkiStatus.PKIConfig}} {{if .Exists}}Создано{{else}}Создайте{{end}} {{.Path}} {{end}}
  Initialize PKI by running "easyrsa init-pki".
Build CA
{{with .pkiStatus.CA}} {{if .Exists}}Создано{{else}}Создайте{{end}} {{.Path}} {{end}}
  Build CA by running "easyrsa build-ca nopass".
Build Server
{{with .pkiStatus.Server}} {{if .Exists}}Создано{{else}}Создайте{{end}} {{.Path}} {{end}}
  Generate a new Server Certificate, Key, and Encryption Files by running "easyrsa gen-req&sign-req server nopas".
Generate DH
{{with .pkiStatus.DH}} {{if .Exists}}Создано{{else}}Создайте{{end}} {{.Path}} {{end}}
  Generate a new Diffie-Hellman key by running "easyrsa gen-dh". This may take up to 10 min. Nothing is freezed, be patient.
Generate TA
{{with .pkiStatus.TA}} {{if .Exists}}Создано{{else}}Создайте{{end}} {{.Path}} {{end}}
  Generate a new openvpn TA key by running "openvpn --genkey --secret ta.key".
INITIALIZE ALL   This button will execute all the steps above one by one - copy EasyRSA vars and create PKI, CA, Server, DH, TA and CRL.
